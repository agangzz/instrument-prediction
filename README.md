# instrument-prediction
There are two instrument prediction models inside this repo. One is trained on MusicNet dataset and one is trained on Musescore dataset.
#### MusicNet model
Contains 7 kinds of instrument - Piano, Violin, Viola, Cello, Clarinet, Horn and Bassoon. Since it's trained on classical music, it will be more correct when testing on classical music.
#### Musescore model
Contains 9 kinds of instrument - Piano, Acoustic Guitar, Electrical Guitar, Drum, Trumpet, Saxphone, Bass, String, Flute. Musecore model is trained on synthetic music. Domain adaptation is still under process to make it perfect.
